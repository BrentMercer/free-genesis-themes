Requirements

The Scholar Theme is a Child Theme of the Genesis Framework, by StudioPress. In order for this theme to work, you must own the Genesis Framework.

I highly recommend the framework, as it's impeccable in almost every corner of development. Slim, beautiful code allows for search engines to easily read your site, allowing for better rankings in search engines like Google and Bing.

The Genesis community is also one of the best I've ever had the privelage of being involved in, so any problems that you have in the future are usually easily and quickly answered if the answer isn't already answered somewhere on the web (highly unlikely).

Required Plugins

The Scholar Theme uses two plugins, Simple Social Icons and Genesis - eNews Extended.

1. Simple Social Icons

Upon installation of the Scholar Theme, you will be asked to install the required plugin, "Simple Social Icons". If you haven't done this, you can do it manually by clicking below:

  • Download Simple Social Icons Plugin

Install this plugin (either through your WordPress Dashboard, or through your FTP Client), and activate it. Then navigate to Appearnce > Widgets.

You'll see a new widget show up with the above title. Drag this over to the Header Right widget area.

Fill out the upper section of the widget that looks like this:

NOTE: If you are using the differnet color schemes (directions here), then change the background color HEX code to correspond with your color scheme. Like so:

  • Default................#E26141
  • Black & White......#333333
  • Olive..................#568153
  • Sky....................#469CCC
  • Fire...................#BE2525
  • Book...................#A27050

After setting these, fill out your social profile URLs, and hit save.

2. Genesis - eNews Extended Plugin

This is an optional plugin, but very useful if you want an easy, and non-obtrusive way to give people a way to sign up to your RSS Feed or Newsletter.

  • Download Simple Social Icons Plugin

Navigation to Appearance > Widgets, and fill out the settings.

Color Schemes

The Scholar Theme comes with 5 additional color schemes to choose from. Please keep in mind that if you are using the Simple Social Icons plugin (info), then you'll want to update the background color to match the color scheme.

You can see a full list of the HEX codes used for each color scheme here.

To set a color scheme, simply navigate to Genesis > Theme Settings. Under Color Style, select the theme that you prefer.

With that being said, I still try my best to make each theme as user-friendly as possible, so you'll see color schemes included quite often, as well as other subtle features.
